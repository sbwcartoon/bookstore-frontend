# Bookstore

## 페이지 및 기능 동작
### 책 목록
- url: /books
- 기능
  - 책 목록 조회
    - 필터링: 제목, 저자 이름, (사용 안함)
    - 페이지네이션(페이지별 항목 10개)
  - 책 목록 표출
    - 표출 정보: 제목, 저자 이름, 수량
- 페이지 이동
  - 새 책 추가 페이지
  - 책 상세 정보 페이지
  - 페이지 이동

### 새 책 추가
- url: /books/new
- 기능
  - 생성: 책 정보 입력 후 저장 버튼 선택
  - 유효성 확인: 저장 전 확인
- 페이지 이동
  - 책 목록 페이지

### 책 상세 정보
- url: /books/:id
- 기능
  - 수정: 책 정보 변경 후 수정 버튼 선택
  - 유효성 확인: 수정 전 확인
  - 삭제: 삭제 후 책 목록 페이지로 자동 이동
- 페이지 이동
  - 책 목록 페이지
  - 아이디에 해당하는 책이 존재하지 않을 경우 책 목록 페이지로 리다이렉트
